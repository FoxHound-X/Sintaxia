
# Sintaxia Learning Apps

Sintaxia Adalah Aplikasi Edukasi Yang di buat Secara Gratis.
Tujuan Di Buatnya Aplikasi Ini adalah untuk membantu Seseorang Yang Ingin
Belajar Pemrograman. Untuk Saat ini kami dari Tim FUZIAN Dev berusaha semaksimal mungkin agar aplikasi Optimal dan Fokus pada pembelajaran

<br>
<br>

## Fitur Dan Fokus

- Memberikan Materi Dasar Pemrograman
- Materi Bahasa Pemrograman dasar seperti HTML, CSS dan JavaScript
- Setelah Kursus akan ada sesi Praktek Langsung
- Support Android dan IOS
- Optimal
